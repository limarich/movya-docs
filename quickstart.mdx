## 🚀 Configuração do Projeto pela Primeira Vez

Siga este guia para instalar e rodar o projeto **Supere App** localmente.

---

## 1️⃣ Pré-requisitos

Antes de começar, certifique-se de ter instalado:

- **Node.js** (>= 18.x)
- **npm** ou **yarn**
- **Git**
- (Opcional) **Android Studio** ou **Xcode** para rodar no emulador

---

## 2️⃣ Clonar o repositório

## 3️⃣ Instalar as dependências

<Accordion title="Instalar dependências" icon="rectangle-terminal">
  Certifique-se de estar na pasta do projeto e execute:
  Com npm:

  ```bash
  npm install
  ```

  Ou com yarn:

  ```bash
  yarn
  ```
</Accordion>

## 4️⃣ Pré-construir o projeto
<Accordion title="Executar prebuild do Expo" icon="hammer"> ```bash npx expo prebuild ``` </Accordion>

## 5️⃣ Rodar o projeto
<AccordionGroup> <Accordion title="Android" icon="mobile-screen"> ```bash yarn android ``` </Accordion> <Accordion title="iOS" icon="mobile"> ```bash yarn ios ``` </Accordion> </AccordionGroup>

Após executar, o projeto rodará no terminal.
Você pode conectar um dispositivo físico ou rodar em um emulador.

## 📦 Estrutura do Projeto
```bash
spr-app/
├── assets/       # Imagens, fontes, ícones e outros arquivos estáticos
├── databases/    # Schemas, Models e Services do Banco de Dados Local
├── tasks(desuso)/# arquivos de configurações do antigo mecanismo de georastreamento
├── components/   # Componentes reutilizáveis da interface
├── dtos/         # Tipagens e estrutura de dados
├── hooks/        # Lógicas reutilizáveis
├── mocks/        # Dados fictícios para testes
├── services/     # Chamadas de API e integrações
├── store/        # Estado global (Zustand, Async Storage, etc.)
├── utils/        # Funções utilitárias
├── app/          # Rotas (Expo Router)

```
<Note> ✅ Agora o projeto está configurado e pronto para desenvolvimento. </Note>
